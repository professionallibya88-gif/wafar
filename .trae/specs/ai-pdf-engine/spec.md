# محرك الذكاء الاصطناعي العالمي لاستخراج البيانات Spec

## Why
نظام استخراج البيانات من ملفات PDF الحالي يعتمد بشكل أساسي على `PyMuPDF` أو `AWS Textract` مع استخدام `Gemini` عبر خدمة Python للبيانات الوصفية فقط. لجعل النظام متقدماً، عالمياً، ودقيقاً، نحتاج إلى دعم عدة نماذج ذكاء اصطناعي (Google, OpenAI, OpenRouter, DeepSeek, Claude) عبر واجهات برمجة التطبيقات (APIs) المباشرة، بحيث يمكن للمستخدم اختيار المزود المناسب سواء كان مدفوعاً أو مجانياً. يجب أيضاً توفير لوحة تحكم متكاملة للإدارة والربط المتقدم.

## What Changes
- **إعدادات النظام**: إضافة مفاتيح الإعدادات الخاصة بمحرك الذكاء الاصطناعي (مزود الخدمة، مفاتيح الـ API، اسم النموذج).
- **الواجهة الخلفية (Backend)**: إنشاء محرك معالجة جديد `AIPDFProcessor.ts` يتصل مباشرة بالمزود المختار من الإعدادات، ويستخرج الجداول والبيانات الوصفية.
- **الواجهة الخلفية (Backend)**: توفير مسارات (Endpoints) للوحة التحكم لإدارة إعدادات الذكاء الاصطناعي.
- **لوحة التحكم (Frontend)**: إنشاء قسم "محرك الذكاء الاصطناعي" يحتوي على قائمة منسدلة (Dropdown) لاختيار المزود (OpenRouter, Google, OpenAI)، حقول لمفاتيح الـ API، وحقل لاسم النموذج.
- **الرفع والمعالجة**: إضافة "محرك الذكاء الاصطناعي العالمي" كخيار في قائمة طرق المعالجة أثناء رفع الملفات أو إعادة معالجتها.

## Impact
- Affected specs: PDF Processing Pipeline, Admin Settings.
- Affected code: `PDFProcessor.ts`, `AIPDFProcessor.ts` (new), `SystemSetting.ts` seeder, Frontend routes/stores, Frontend Upload/Settings views.

## ADDED Requirements
### Requirement: محرك ذكاء اصطناعي متعدد المزودين
يجب أن يوفر النظام آلية لاستخراج البيانات (الجداول والبيانات الوصفية) باستخدام الذكاء الاصطناعي.

#### Scenario: معالجة ملف PDF
- **WHEN** يقوم المشرف برفع ملف واختيار "محرك الذكاء الاصطناعي".
- **THEN** يقوم النظام بقراءة المزود المختار من الإعدادات (مثلاً OpenRouter) والمفتاح واسم النموذج، ويرسل محتوى الـ PDF للتحليل واستخراج البيانات بدقة عالية.

### Requirement: قسم لوحة تحكم محرك الذكاء الاصطناعي
يجب أن يتمكن المشرف من ضبط إعدادات المحرك من لوحة التحكم.

#### Scenario: ضبط الإعدادات
- **WHEN** يدخل المشرف إلى قسم "محرك الذكاء الاصطناعي".
- **THEN** تظهر له واجهة تحتوي على قائمة المزودين وحقول المفاتيح وإمكانية الحفظ والاختبار.
