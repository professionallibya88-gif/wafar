# المهام

- [x] المهمة 1: تحليل تدفق الرفع الحالي وتحويله إلى رفع دفعات.
  - [x] المهمة الفرعية 1.1: تحديد نقاط التحويل من ملف واحد إلى عدة ملفات في الواجهة وواجهة API والخلفية.
  - [x] المهمة الفرعية 1.2: اعتماد صيغة طلب موحدة لرفع عدة ملفات مع بيانات مستقلة لكل ملف.

- [x] المهمة 2: بناء واجهة رفع متعددة الملفات مع بيانات مستقلة لكل ملف.
  - [x] المهمة الفرعية 2.1: تحديث شاشة الرفع لتسمح باختيار عدة ملفات وعرضها في قائمة واضحة.
  - [x] المهمة الفرعية 2.2: تنفيذ استخراج أولي للبيانات لكل ملف وإتاحة تعديل المورد والتاريخ والحقول المطلوبة قبل الإرسال.
  - [x] المهمة الفرعية 2.3: عرض حالات التحقق والأخطاء على مستوى كل ملف بدل الخطأ العام فقط.

- [x] المهمة 3: تنفيذ رفع ومعالجة وحفظ مستقلين لكل ملف في الدفعة.
  - [x] المهمة الفرعية 3.1: تحديث مسارات الرفع والمتحكمات لتستقبل عدة ملفات في الطلب الواحد.
  - [x] المهمة الفرعية 3.2: تحديث الخدمة والمعالجة والطابور لحفظ كل ملف كسجل مستقل ومعالجة النجاح الجزئي والفشل الجزئي.
  - [x] المهمة الفرعية 3.3: إرجاع استجابة منظمة تحتوي نتائج كل ملف وحالته ومعرّفه ورسائل الخطأ الخاصة به إن وجدت.

- [x] المهمة 4: عرض التقدم والنتائج لكل ملف بعد بدء الرفع.
  - [x] المهمة الفرعية 4.1: بناء تتبع تقدم مستقل لكل ملف داخل الواجهة.
  - [x] المهمة الفرعية 4.2: إظهار الملفات المكتملة والفاشلة والتي ما زالت قيد المعالجة بشكل واضح.
  - [x] المهمة الفرعية 4.3: دعم إعادة المحاولة للملفات الفاشلة من دون المساس بالملفات الناجحة.

- [x] المهمة 5: تمييز الملفات الجديدة داخل `ملفاتي`.
  - [x] المهمة الفرعية 5.1: تحديد قاعدة التمييز الزمني للملفات الجديدة اعتمادًا على وقت الإنشاء.
  - [x] المهمة الفرعية 5.2: إضافة شارة أو لون واضح للملفات الجديدة داخل القائمة.
  - [x] المهمة الفرعية 5.3: إزالة التمييز تلقائيًا بعد انتهاء المدة المحددة مع بقاء البيانات الأخرى كما هي.

- [x] المهمة 6: التحقق النهائي من السلوك الكامل للميزة.
  - [x] المهمة الفرعية 6.1: اختبار رفع عدة ملفات ناجحة في دفعة واحدة.
  - [x] المهمة الفرعية 6.2: اختبار دفعة مختلطة تحتوي ملفات ناجحة وأخرى فاشلة.
  - [x] المهمة الفرعية 6.3: اختبار حفظ البيانات المستقلة لكل ملف وظهورها في `ملفاتي`.
  - [x] المهمة الفرعية 6.4: اختبار ظهور تمييز "جديد" واختفائه بعد انتهاء المدة الزمنية.
