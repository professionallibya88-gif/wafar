# لوحة التحكم - دليل المطور

هذا المجلد يحتوي على جميع صفحات لوحة التحكم (Admin Panel) للنظام.

## الصفحات المتاحة

### 1. AdminDashboardView

لوحة الإحصائيات الرئيسية تعرض:

- إحصائيات المستخدمين
- إحصائيات الموردين
- إحصائيات الملفات
- إحصائيات المدفوعات المعلقة
- المستخدمين الأخيرين
- المدفوعات المعلقة
- الملفات الأخيرة

### 2. AdminUsersView

إدارة المستخدمين:

- قائمة المستخدمين مع Pagination
- فلترة حسب الدور (مدير، تاجر، مورد)
- تفعيل/تعطيل المستخدمين
- حذف المستخدمين
- عرض عدد المستخدمين النشطين والمديرين

### 3. AdminSuppliersView

إدارة الشركات الموردة:

- قائمة الشركات مع Pagination
- فلترة حسب الفئة
- البحث في الشركات
- إضافة شركة جديدة
- تعديل بيانات الشركة
- حذف الشركة

### 4. AdminPaymentsView

إدارة المدفوعات:

- قائمة المدفوعات مع Pagination
- فلترة حسب الحالة (معلق، مقبول، مرفوض)
- قبول المدفوعات
- رفض المدفوعات
- عرض إجمالي المبالغ المقبولة

### 5. AdminPlansView

إدارة باقات الاشتراك:

- عرض جميع الباقات
- إنشاء باقة جديدة
- تعديل باقة موجودة
- حذف باقة
- عرض الميزات والحدود لكل باقة

### 6. AdminFilesView

إدارة ملفات PDF:

- قائمة الملفات مع Pagination
- فلترة حسب الحالة
- البحث في الملفات
- إعادة معالجة الملفات الفاشلة
- حذف الملفات

### 7. AdminSettingsView

إعدادات النظام:

- الإعدادات العامة (اسم الموقع، الوصف، الخط)
- إعدادات معالجة PDF
- إعدادات البريد الإلكتروني (SMTP)
- إعدادات الأمان
- إعدادات النظام (المنطقة الزمنية، الصيانة)

### 8. AdminNotificationsView

إدارة الإشعارات:

- إنشاء إشعار لمستخدم محدد
- بث إشعار لجميع المستخدمين
- إرسال إشعار لمجموعة من المستخدمين

### 9. AdminMonitoringView

مراقبة النظام الأساسية:

- حالة النظام
- قائمة المعالجة
- استخدام الذاكرة والمعالج
- تفاصيل الصحة
- التنبيهات
- تقدم المعالجة
- تحديث تلقائي قابل للتعديل

### 10. AdminAdvancedMonitoringView

مراقبة النظام المتقدمة:

- رسوم بيانية للأداء
- موارد النظام (CPU, RAM, Disk)
- المهام الحالية
- أزرار الإجراءات السريعة (مسح المكتملة، إعادة الفاشلة، تصدير التقرير)
- تحديث تلقائي قابل للتعديل

### 11. AdminActivityLogsView

سجلات النشاط:

- سجل جميع الأنشطة في النظام
- فلترة حسب نوع الإجراء (إنشاء، تعديل، حذف، تسجيل دخول، تسجيل خروج)
- البحث في السجلات
- عرض المستخدم، الإجراء، التفاصيل، IP، التاريخ
- Pagination

## Composables الموحدة

يتم استخدام composables موحدة في جميع الصفحات لضمان التوحيد:

- `usePagination`: إدارة الصفحات

## API Endpoints

جميع الصفحات تستخدم `adminAPI` من `@/services/api`:

```javascript
import { adminAPI } from "@/services/api";
```

## قواعد التطوير

1. **اللغة**: جميع النصوص بالعربية الفصحى
2. **التصميم**: RTL مع TailwindCSS
3. **الأيقونات**: Heroicons فقط (لا إيموجي)
4. **الأرقام**: أرقام عربية (123) وليس هندية (١٢٣)
5. **الأخطاء**: معالجة الأخطاء بالعربية عبر Toast
6. **Pagination**: جميع القوائم تحتوي على Pagination
7. **الفلاتر**: جميع القوائم تحتوي على فلتر مناسب
8. **التوحد**: استخدام composables الموحدة

## إضافة صفحة جديدة

لإضافة صفحة جديدة للوحة التحكم:

1. إنشاء الملف في `src/views/admin/YourNewView.vue`
2. إضافة المسار في `src/router/index.js`
3. إضافة الرابط في `src/layouts/AdminLayout.vue`
4. استخدام composables الموحدة
5. اتباع قواعد التطوير المذكورة أعلاه
